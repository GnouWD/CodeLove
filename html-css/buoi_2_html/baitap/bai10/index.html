Đây là trang web của tôi

Chào mừng đến với trang web của tôi. Đây là một ví dụ về cách sử dụng thẻ image để hiển thị hình ảnh trên trang web.

Trên đây là hình ảnh ngẫu nhiên được lấy từ google. Bạn có thể thay đổi kích thước hình ảnh bằng cách chỉnh độ rộng và chiều cao trong thuộc tính'width' hoặc 'height' của thẻ. Bạn cũng có thể thêm một mô tả bằng thuộc tính alt.
